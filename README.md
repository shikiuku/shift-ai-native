# シフトai (Shift AI) - Mobile Edition 📱🌿

「シフトai」は、Expo (React Native) を使用して構築された、モバイル・タブレット向けの高度なシフト管理システムです。

## 🌟 特徴

- **🤖 AI自動シフト生成**: スタッフの希望と相性を考慮した人員配置。
- **📱 モバイルファースト**: スマホでの操作性を重視したUIデザイン。
- **💻 タブレット対応**: 広い画面を活かしたサイドバーと横画面表示。
- **🎨 Green Pack Design**: 目に優しく清潔感のある和のモダンデザイン。

## 🚀 はじめかた

### 必要な環境
- Node.js (v18以上)
- Expo Go アプリ (スマホでの動作確認用)

### セットアップ手順

1. **依存関係のインストール**:
   ```bash
   npm install
   ```

2. **アプリの起動**:
   ```bash
   npx expo start
   ```

3. **動作確認**:
   - 画面に表示されるQRコードをスマホの Expo Go アプリで読み取ってください。
   - または `w` を押してWebブラウザで確認できます。

## 🛠 技術構成
- **Framework**: Expo (SDK 54+)
- **UI**: React Native, Lucide React Native
- **Logic**: TypeScript (AI生成アルゴリズム)
